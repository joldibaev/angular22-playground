Mask

Lightweight input formatting for common patterns and grouped decimal values, with Signal Forms and classic Angular Forms integration.

Pattern tokens

Add uiMask to a text input. Literal characters are inserted automatically; token characters accept and normalize user input.

Behavior
SettingBehavior
separator.N Groups the integer part and accepts at most N decimal digits. Use thousandSeparator and decimalMarker to choose the characters.
Model value Forms receive the unmasked value. Group separators and pattern literals are removed; the configured decimal marker is retained.
Validation A mask formats input but does not make it valid or complete. Add validators for required length, ranges, and business rules.
Input type Use type="text" with an appropriate inputmode. The directive intentionally does not attach to type="number" inputs.
